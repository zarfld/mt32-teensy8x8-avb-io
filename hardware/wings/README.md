# Wing Boards

This directory contains notes and future hardware files for the analog wing boards.

Wing boards attach to the Teensy8x8AudioBoard IDC connectors and provide the physical audio I/O connectors (TRS, XLR, Combo).

## Available Wing Variants

| Wing | Type | File |
|---|---|---|
| TRS Input | Balanced line-level input | [`trs-input.md`](trs-input.md) |
| TRS Output | Balanced line-level output | [`trs-output.md`](trs-output.md) |
| XLR Input | Balanced line-level input | [`xlr-input.md`](xlr-input.md) |
| XLR Output | Balanced line-level output | [`xlr-output.md`](xlr-output.md) |

## Population Notes

See [`docs/wing-board-population.md`](../../docs/wing-board-population.md) for detailed population tables.

## Status

Wing board hardware files are not included in this repository (they belong to the upstream `Teensy8x8AudioBoard` project, see `external/Teensy8x8AudioBoard`). This directory holds integration notes specific to this project.
